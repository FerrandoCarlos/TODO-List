# TODO List 📝

---

Aplicación de lista de tareas desarrollada por MILODEV

## Tecnologías usadas

---

- Express
- Pug
- Tailwind
- Zod
- Ajax

## Instalación

---

1. Clonar el repositorio

```bash
git clone https://github.com/FerrandoCarlos/TODO-List.git
cd TODO-List
```

2. Instalar dependencias

```bash
npm install
```

3. Crear el archivo `.env` en la raíz
   - PORT=3000
4. Iniciar el servidor

```bash
nodemon
```

5. Abrir en el navegador
   - http://localhost:3000
